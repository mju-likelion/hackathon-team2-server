## 📠 Convention

### 🤝 Branch Naming Convention

|  머릿말      | 설명        |
| ----------- | ---------- |
| master      | 서비스 브랜치    |
| feature     | 기능 단위 구현    |
| hotfix      | 서비스 중 긴급 수정 건에 대한 처리   |

### 🤝 Commit Convention

|  머릿말     | 설명        |
| ----------- | ---------- |
| feat        | 기능 구현, 추가   |
| setting     | 패키지 설치, 개발 설정    |
| refactor    | 코드 리팩터링    |
| fix         | 버그 수정, 예외 케이스 대응, 기능 개선   |
| docs        | README.md 작성, 주석 작성   |
| chore       | 기타 작업  |
